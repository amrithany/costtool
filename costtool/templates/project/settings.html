CBCSE Cost Tool Kit includes a number of default settings that you have the option to change by clicking on the button.

  • Discount rate used to calculate Net Present Value of costs incurred after Year 1 (Default =3.5%)
  • Year in which you want to express costs (Default = latest available year for which we have an inflation index)
  • Geographical location in which you want to express costs (Default=All States, All Areas)
  • Database of prices to use and filters (Default = all prices in CBCSE Database of Educational Resource Prices and in MyPrices Database)
  • Hours per year used in Wage Converter
